# Still — catalog block (test assignment)

Pure **HTML + CSS** catalog section with light **vanilla JS** (filters, search, sort, View modal). Visual system from pet project [Still Store](https://elenasamanchuk.github.io/still-store/). Product data and images bundled from Fake Store API (`products.json` + `img/`).

- 8 product cards
- No UI kits, no JS, no build step
- Live: https://elenasamanchuk.github.io/shop-catalog-block/
